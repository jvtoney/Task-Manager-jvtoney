# Task Manager
A simple task manager built in Android Studio with Kotlin and Jetpack Compose

---

## Features
- Text Input Field
- Submit Button
- Check List after submissions

---

## Technology Used
- Android Studio
- Kotlin
- Jetpack Compose

---

## Running the App

### 1. Clone the Repository
```bash
git clone https://github.com/jvtoney/Task-Manager-jvtoney.git
cd Task-Manager-jvtoney
```

### 2. Open in Android Studio
- Select **Open an Existing Project**
- Let Gradle sync automatically
